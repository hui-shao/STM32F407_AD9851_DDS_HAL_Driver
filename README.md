硬件连接如下表（也可以对照 CubeMX）：

| AD985x模块管脚 | 单片机管脚 | 备注                                                     |
| -------------- | ---------- | -------------------------------------------------------- |
| AD985x_RST     | PD4        |                                                          |
| AD985x_CLK     | PD0        |                                                          |
| AD985x_FQ      | PD5        |                                                          |
| AD985x_D7      | PD3        | 本代码使用串行模式进行驱动，只使用 D7 ，连接到单片机 PD3 |
| 5V             | 5V         |                                                          |
| GND            | GND(0V)    | 不要忘了共地                                             |

注：

1. 原版程序中，如果使用并行模式驱动，D0-07 分别连接到 PD0-PD7（按需修改）。
2. 使用串行模式时，D0、D1、D2 的电平建议通过电路配置为 `110`。
